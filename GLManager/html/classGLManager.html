Creates a buffer of any type. The parameters are as follows: vbo: GLuint& pointer to buffer. vbuffer: void* Pointer to data of the buffer buffSize: int Buffer size

target: GL_ARRAY_BUFFER, GL_ELEMENT_ARRAY_BUFFER, GL_PIXEL_PACK_BUFFER, GL_PIXEL_UNPACK_BUFFER

usage: GL_STREAM_DRAW, GL_STREAM_READ, GL_STREAM_COPY, GL_STATIC_DRAW, GL_STATIC_READ, GL_STATIC_COPY, GL_DYNAMIC_DRAW, GL_DYNAMIC_READ, GL_DYNAMIC_COPY

vAtribIndx: index of the atribute on the shader size: Size of each 'cell' in the buffer. If it is a color it should be 4 RGBA norm: If the values are normalized [0:1] stride: Offset between consecutive genertic vertex attributes ponter: Offset of the first component fo the array

type: GL_BYTE, GL_UNSIGNED_BYTE, GL_SHORT, GL_UNSIGNED_SHORT, GL_INT, GL_UNSIGNED_INT, GL_FLOAT, GL_DOUBLE
